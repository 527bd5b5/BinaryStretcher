Binary Stretcher 1.0.0

◆ これは何？

これは冗長性を持つ（単純にバイナリーを繰り返した）ファイルを生成・復元するプログラムです。

◆ 使用例

「-e」オプションで引数に指定されたファイルからBSXファイルを同じ階層に作成します。引数には複数のファイルを指定することができます。

java -jar BinaryStretcher.jar -e hoge.txt fuga.png piyo.zip

「-d」オプションで引数に指定されたBSXファイルの検証と修復を行い、元のファイルを同じ階層に作成します。エンコードの直後に実行した場合は元のファイルが上書きされるため、安全に処理するためには「--dry-run」オプションで結果を確認した後にデコードすることが推奨されます。

java -jar BinaryStretcher.jar -d hoge.txt.bsx fuga.png.bsx piyo.zip.bsx

◆ オプション

-e
ファイルからBSXファイルを作成します（エンコード処理）。

-d
BSXファイルから元のファイルを作成します（デコード処理）。

-m <magnification: int = 4>
エンコード時の倍率を指定します。デコード時はBSXファイルから読み取られます。

-b <buffered_size: int = 1024>
一度にバッファリングするファイルのサイズ（バイト数）を指定します。エンコード時は入力ファイル、デコード時は出力ファイルに割り当てられます。

--extention <extention: string = "bsx">
BSXファイルの拡張子（ピリオドを除く）を指定します。

--dry-run
ファイルを作成せずに結果のみを表示します。デコード時に指定することでBSXファイルの修復のみを行います。

--no-fix
デコード時にBSXファイルを修復しません。

--log-level <level: int = 4>
標準出力されるログの種類を指定します。
0: 何も表示しない
1: 致命的なエラーのみを表示する
2: ファイルの状況と結果を表示する
3: エラー時に詳細な結果を表示する
4: 常に詳細な結果を表示する

-v
--version
プログラムのバージョンを表示します。

-h
--help
プログラムの使い方を表示します。
